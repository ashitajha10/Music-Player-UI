# Spotify Clone - Responsive Web Player

A static clone of the Spotify Web Player built using pure **HTML** and **CSS**. This project replicates the modern and sleek UI of Spotify, providing a premium music streaming experience across all devices.

## ✨ Features
- **Mobile Responsive**: Fully optimized for mobile (≤480px), tablet (481px–768px), laptops, and large desktops.
- **Dynamic Layout**: 
  - Sidebar collapses to a minimalist icon-view on tablets.
  - Mobile bottom navigation bar for a native app feel.
- **Modern UI**: Implements Spotify's dark theme, Montserrat typography, and card-based grid layout.
- **Music Player Bar**: Fixed bottom player with simplified controls for small screens.
- **Pure CSS**: No JavaScript used, leveraging Flexbox and CSS Media Queries for all layout transitions.

## 🛠️ Built With
- **HTML5**: Semantic structure.
- **CSS3**: Core styling, Flexbox, and Media Queries.
- **Font Awesome**: Icons for navigation and player controls.
- **Google Fonts**: Montserrat for secondary and UI text.

## 📱 Responsiveness Overview
- **Large Screens**: Full sidebar with library boxes and wide card grids.
- **Tablets**: Narrow sidebar (icons only), optimized padding for scrolling.
- **Mobile Devices**: Hidden sidebar, bottom navigation, and stacked card grid for better focus.

## 🏁 How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/ashitajha10/Spotify-Clone.git
   ```
2. Open `index.html` in any modern web browser.
